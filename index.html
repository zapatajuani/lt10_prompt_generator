LT-10 Prompter

Most behaviors of LT-10 could be changed by Configuration Parameters. You could change the setting of configuration parameters by GlobalSat Configuration App or by sending downlink command.

Device

O4

Power on operating mode

Other Settings

Gt

G-sensor sensitivity for motion detection

O1

Interval for triggering motion sensor

Valor incorrecto. O1 debe estar entre 1 ~ 100

Z5

Device Security Key for App (8 char long)

Valor incorrecto. Longitud maxima de 8 caracteres

Z6

SPM (Smart Power Management Level)

GPS

CD

enable/disable GPS

C1

The time to get GPS-fix if LT-10 got GPS-fix over 1 hour ago

Valor incorrecto. C1 debe estar entre 30 ~ 600

C2

The time to get GPS-fix if LT-10 LT-10 got GPS-fix within 1 hour

Valor incorrecto. C2 debe estar entre 10 ~ 120

Communication

A1

Wait confirmation from gateway after sending message to gateway

A6

Number of re-sending reports without getting ACK from gateway

Tracking - Period

P0

Report interval of period report

Valor incorrecto. El valor debe ser mayor a 600

Tracking - Motion

R0

Report interval in static state

Valor incorrecto. El valor debe ser mayor a 600

R1

Report interval in moving state

Valor incorrecto. El valor debe ser mayor a 600
